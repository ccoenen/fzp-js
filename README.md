# fzp-js [![](https://paulvollmer.net/fzp-js/badge.svg)](https://paulvollmer.net/fzp-js/)

fritzing fzp javascript library

## Development
```sh
yarn install
make test
```

and lint your files before you commit
```sh
make lint
```

to build an es5 compatible version run
```sh
make build
```
